## 収入
```dataview
TABLE WITHOUT ID
    file.name AS "費目",
    date.day AS "日付",
    amount AS "金額"
FROM #収入
WHERE date.year = 2023 AND date.month = 9
SORT date ASC
```
```dataview
TABLE WITHOUT ID
    sum(rows.amount) AS "合計"
FROM #収入
WHERE date.year = 2023 AND date.month = 9
GROUP BY true
```
