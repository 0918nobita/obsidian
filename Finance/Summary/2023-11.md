## 収支
```dataviewjs
await dv.view('scripts/monthly-expenses', { year: 2023, month: 11 });
```
### 支出
```dataview
TABLE WITHOUT ID
    file.name AS "費目",
    date.day AS "日付",
    amount AS "金額"
FROM #支出
WHERE date.year = 2023 AND date.month = 11
SORT date ASC
```
```dataview
TABLE WITHOUT ID
    sum(rows.amount) AS "合計"
FROM #支出
WHERE date.year = 2023 AND date.month = 11
GROUP BY true
```
